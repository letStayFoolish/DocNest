## Navigation

| Action | Windows / Linux | macOS |
| --- | --- | --- |
| Go to declaration | `Ctrl + B` | `Cmd + B` |
| Go to implementation | `Ctrl + Alt + B` | `Cmd + Opt + B` |
| Find usages | `F7` | `F7` |
| View recent files | `Ctrl + E` | `Cmd + E` |
| Project window | `Alt + 1` | `Cmd + 1` |
| Focus code editor | `Esc` | `Esc` |
| Search everywhere | `Shift` twice | `Shift` twice |
| Find action | `Shift + Ctrl + A` | `Shift + Cmd + A` |
| Run anything | `Ctrl` twice | `Ctrl` twice |

## Editing

| Action | Windows / Linux | macOS |
| --- | --- | --- |
| Show suggestions | `Alt + Enter` | `Alt + Enter` |
| Next error | `F2` | `F2` |
| Extend selection | `Ctrl + W` | `Opt + ArrowUp` |
| Shrink selection | `Shift + Ctrl + W` | `Shift + Opt + ArrowUp` |
| Complete current statement | `Shift + Ctrl + Enter` | `Shift + Cmd + Enter` |
| Reformat file | `Ctrl + Alt + L` | `Opt + Cmd + L` |
| Refactor this | `Shift + Ctrl + Alt + T` | `Ctrl + T` |
